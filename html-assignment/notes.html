5 HTML Tags I've Learned

Strong
Strong is a tag that can be used to bold text.
P
P, paragraph, is a tag used to show that whatever comes after it will be a paragraph.
DL
The dl, description list, tag is a tag used to create a list where you can add a description to.
OL
The ol, ordered list, tag is used to create an ordered list with bullet points.
UL
The ul, unordered list, tag is used to created unordered lists.
